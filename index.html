Android Plugin DSL Reference

This page provides reference documentation for the Android Gradle Plugin DSL. For instructions on how to work on the plugin itself, please see Building the Android Gradle Plugin. If you encounter issues using the plugin or Android Studio, please go to our public issue tracker and choose an appropriate component from the dropdown menu.

To see the reference documentation, choose the plugin version:
